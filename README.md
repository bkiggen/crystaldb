## About The Project

CrystalDB provides a number of features to managers and employees of subscription box companies. It tracks historical data and uses that information to make suggestions for upcoming shipments. This allows for easy planning and building of subscription boxes which don't repeat contents for long-term subscribers. This app also features a ShipDay display, which supports employees on the day of physically shipping packages, directing them to build shipments based on the previously compiled lists.
